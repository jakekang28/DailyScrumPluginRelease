BeCalm: AI-Work Digest

이용약관 및 개인정보 처리방침

시행일: 2026-02-12 운영자: Without 문의: w.out2026@gmail.com

본 문서는 크롬 확장프로그램 BeCalm: AI-Work Digest(이하 "서비스")의 이용조건 및 개인정보 처리에 관한 사항을 규정합니다. 사용자는 서비스 설치·실행 및 "동의" 버튼 클릭(또는 이에 준하는 동작)을 통해 본 약관 및 개인정보 처리방침에 동의합니다.

1. 용어 정의

  1. 서비스: 회사가 제공하는 크롬 확장프로그램 및 관련 서버/웹 기능 일체
  2. 사용자: 서비스를 설치·접속·이용하는 자
  3. 원문 데이터: 보고서 생성 전 단계에서 수집·처리되는 텍스트/메타데이터(예: AI 대화 원문, URL 등)
  4. 보고서: 서비스가 자동 생성·제공하는 Daily Scrum 요약 결과물
  5. 처리위탁(수탁사): 회사로부터 개인정보 처리업무를 위탁받아 처리하는 외부 사업자(예: 클라우드, AI API 제공자)

2. 약관의 효력 및 변경

  1. 본 약관은 사용자가 동의 후 서비스를 이용하는 시점부터 효력이 발생합니다.
  2. 회사는 관련 법령을 위반하지 않는 범위에서 약관/방침을 변경할 수 있으며, 변경 시 서비스 내 공지, 웹페이지 공지, 이메일 안내 등 합리적인 방식으로 고지합니다.
  3. 변경 이후에도 사용자가 서비스를 계속 이용하면 변경 약관에 동의한 것으로 봅니다.

3. 서비스의 내용

서비스는 사용자의 업무/작업 맥락을 기반으로 다음 기능을 제공합니다.

  • Daily Scrum 보고서 자동 생성(업무 요약, 작업 로그 구조화, 인지 흐름 기반 정리 등)
  • 사용자의 작업 기록을 기반으로 한 정리/요약 결과물 제공

4. 서비스 이용 및 권한

  1. 서비스는 기능 제공을 위해 브라우저 권한 및 외부 서비스(예: Google Workspace, Notion, Slack 등) 접근 권한을 요청할 수 있습니다.
  2. 회사는 기능 수행에 필요한 최소 권한만 요청하며, 데이터 처리 목적과 범위를 사용자에게 고지합니다.
  3. 사용자는 언제든지 브라우저 설정에서 권한을 철회하거나 확장프로그램을 제거하여 데이터 수집을 중단할 수 있습니다.

5. 사용자 의무 및 금지행위

사용자는 다음 행위를 해서는 안 됩니다.

  • 서비스/시스템을 역설계, 취약점 공격, 비정상 트래픽 유발 등으로 방해하는 행위
  • 타인의 계정/데이터에 무단 접근하는 행위
  • 법령 또는 타인의 권리를 침해하는 방식으로 서비스를 이용하는 행위

6. 지식재산권

  1. 서비스 및 관련 소프트웨어/브랜드/UI/템플릿 등의 권리는 회사 또는 정당한 권리자에게 있습니다.
  2. 사용자가 제공한 원본 콘텐츠(원문 데이터)의 권리는 사용자에게 귀속됩니다.
  3. 회사는 보고서 생성 및 제공을 위해 필요한 범위에서 사용자의 원문 데이터를 처리할 수 있습니다.

7. 서비스 중단 및 해지

  1. 회사는 서비스 운영상 필요한 경우(점검, 장애 대응, 시스템 변경 등) 일시적으로 서비스를 중단할 수 있습니다.
  2. 사용자는 언제든지 확장프로그램 제거 또는 기능 비활성화로 서비스 이용을 중단할 수 있습니다.

8. 면책 및 책임 제한 (베타/MVP 고지)

  1. 서비스는 베타/MVP 단계로 일부 기능 오류, 요약 결과의 누락·부정확성이 발생할 수 있습니다.
  2. 회사는 관련 법령이 허용하는 범위에서 간접손해, 특별손해 등에 대한 책임을 제한할 수 있습니다.
  3. 단, 회사의 고의 또는 중대한 과실로 인한 손해는 예외로 합니다.

9. 준거법 및 관할

본 약관은 대한민국 법령을 준거법으로 하며, 분쟁 발생 시 민사소송법상 관할 법원에 따릅니다.

개인정보 처리방침

10. 개인정보 처리의 원칙

회사는 개인정보 관련 법령을 준수하며, 보고서 생성에 필요한 최소한의 데이터만 수집·이용합니다. 사용자는 동의를 거부할 수 있으며, 동의하지 않으면 서비스의 핵심 기능(보고서 자동 생성 등)이 제한될 수 있습니다.

11. 수집하는 데이터

11.1 수집하는 데이터 (사용자 동의 시)

회사는 사용자가 동의한 경우, Daily Scrum 보고서 생성을 위해 다음 데이터를 수집·처리할 수 있습니다.

  1. AI 도구 대화 내용(원문) — ChatGPT, Claude, Gemini 등에서의 질문과 응답(사용자가 해당 기능을 활성화한 경우)
  2. 협업 도구 활동(메타데이터/요약) — Notion: 페이지 제목 등(연동/권한 승인 시), Slack: 메시지 요약 등(연동/권한 승인 시)
  3. Google Workspace 활동(메타데이터) — Docs/Sheets/Slides 문서 제목 및 편집 활동 메타데이터(예: 문서 식별자, 수정 이벤트 시각, 변경 여부 등)
  4. 웹 브라우징 데이터 — 접속 Website URL(전체 URL): 사용자가 방문한 페이지의 전체 URL
  5. 인터랙션 메트릭(집계 지표) — 클릭 횟수, 스크롤 패턴(구간/빈도/방향 등), 타이핑 컨텍스트(입력 "내용" 제외: 입력 여부/분량/속도 등), 탭 전환 패턴, 집중도 분포 등

11.2 수집하지 않는 데이터

회사는 아래 정보를 수집하지 않습니다.

  • 비밀번호, 신용카드 정보 등 민감 정보
  • 키 입력 내용(키로깅 없음)
  • 복사/붙여넣기 한 "내용"
  • 스크롤 좌표, 클릭 위치 등 정밀 좌표/포지션

사용자가 AI 대화/문서 등에 민감정보를 직접 입력하는 경우, 그 "입력된 원문"이 보고서 생성 과정에 포함될 수 있으므로 사용자는 민감정보 입력에 유의해야 합니다.

12. 개인정보 이용 목적

회사는 수집된 데이터를 오직 다음 목적에만 사용합니다.

  1. Daily Scrum 보고서 생성 및 제공(요약/구조화/정리 결과 제공)
  2. 서비스 품질 개선을 위한 최소 범위의 오류/장애 대응 및 보안 로그 관리

회사는 광고 타게팅, 데이터 판매, 데이터 브로커 제공 등 보고서 생성과 무관한 목적으로 데이터를 이용하지 않습니다.

13. 처리의 법적 근거(동의)

  1. 회사는 사용자의 명시적 동의를 근거로 원문 데이터를 수집·이용·외부 전송을 수행합니다.
  2. 사용자는 동의를 거부할 수 있으며, 이 경우 핵심 기능(보고서 자동 생성 등)이 제한될 수 있습니다.
  3. 사용자는 언제든지 설정에서 기능 비활성화 또는 확장프로그램 제거로 동의를 철회할 수 있습니다.

14. 외부 전송 및 처리(Claude API 사용)

회사는 보고서 생성(요약/구조화)을 위해 사용자가 동의한 원문 데이터를 외부 AI API(Anthropic Claude API)로 전송하여 처리할 수 있습니다.

  • 전송되는 데이터(예시): AI 도구 대화 원문(활성화 시), 협업/워크스페이스 제목 및 활동 메타데이터(연동 시), 전체 URL
  • 전송 목적: Daily Scrum 보고서 생성을 위한 요약/분류/구조화 처리
  • 전송 시점: 사용자가 보고서 생성을 실행하거나 자동 생성 트리거가 발생하는 경우
  • 전송 범위: 보고서 생성에 필요한 최소 범위

사용자는 언제든지 설정에서 관련 기능을 비활성화하거나 확장프로그램을 제거하여 외부 전송을 중단할 수 있습니다.

15. 보관 기간 및 파기

  1. 회사는 원칙적으로 개인정보 처리 목적이 달성되면 지체 없이 파기합니다.
  2. 다만 사용자가 서비스 이용을 지속하는 동안, 회사는 서비스 제공 및 보고서 히스토리 제공을 위해 원문 데이터를 계정 삭제 직전까지 보관할 수 있습니다.
  3. 사용자가 계정을 삭제한 경우, 회사는 계정 삭제일로부터 30일 이내 개인정보를 파기합니다.
  4. 파기 방법: 전자적 파일은 복구 불가능한 방식으로 삭제합니다.

16. 데이터 보안

회사는 개인정보의 안전성 확보를 위해 합리적인 보호조치를 시행합니다.

  • 전송 구간 암호화: TLS 등 안전한 통신 적용
  • 암호화: 데이터는 AES-GCM-256 등 강력한 방식으로 보호되도록 설계·운영합니다(전송/저장 과정 포함)
  • 저장 및 접근 통제: Supabase 등 서버에 안전하게 저장되며, 사용자 인증 및 DB 접근제어(예: RLS)로 본인만 접근 가능하도록 설계합니다
  • 접근 권한 최소화, 접근 로그 관리, 보안 업데이트 등 운영적 보호조치

17. 제3자 제공 및 처리위탁

17.1 제3자 제공

회사는 사용자의 원문 데이터를 제3자에게 판매하거나 광고 목적으로 제공하지 않습니다.

17.2 처리위탁

회사는 서비스 제공을 위해 아래 업체에 개인정보 처리업무를 위탁할 수 있으며, 위탁계약을 통해 개인정보 보호 의무를 부과하고 감독합니다.

  1. Anthropic (Claude API) — 위탁업무: 텍스트 요약/분류/구조화 등 보고서 생성을 위한 AI 처리. 처리 데이터: 사용자가 동의한 원문 데이터(요약에 필요한 범위, 전체 URL 포함 가능)
  2. Supabase — 위탁업무: 사용자 인증, 데이터베이스 저장/조회, 서비스 운영 인프라. 리전/국가: Southeast Asia(Singapore)

수탁사가 추가/변경되는 경우 회사는 본 방침을 업데이트하고 고지합니다.

18. 개인정보 국외이전(처리위탁) 고지 및 동의

회사는 Claude API 사용을 위해 개인정보를 국외로 이전(처리위탁)할 수 있습니다.

18.1 국외이전 내역

  • 이전받는 자(수탁사): Anthropic (Claude API 제공자)
  • 이전 국가: 미국(Anthropic 본사 소재지)
  • 이전 일시/방법: 보고서 생성 실행 또는 자동 생성 트리거 발생 시, 네트워크를 통한 API 전송
  • 이전 항목: 사용자가 동의한 원문 데이터(AI 도구 대화 원문, 제목/활동 메타데이터, 전체 URL)
  • 이용 목적: Daily Scrum 보고서 생성(요약/분류/구조화)
  • 보유/이용 기간: 수탁사의 정책 및 계약에 따르며, 회사는 필요 최소 기간 내 처리되도록 관리합니다.

18.2 국외이전 동의 거부 및 영향

  • 사용자는 국외이전에 동의하지 않을 수 있습니다.
  • 거부 시 Claude API 기반 요약/보고서 자동 생성 기능 제공이 제한될 수 있습니다.

20. 개인정보 처리방침의 변경

회사는 법령 또는 서비스 변경사항을 반영하기 위해 본 방침을 변경할 수 있으며, 변경 시 서비스 내 공지 또는 이메일 등 합리적인 방식으로 고지합니다.

데이터 수집 및 사용 동의 요약

  • 수집: AI 대화 원문, 협업/워크스페이스 제목·활동 메타데이터, 전체 URL, 집계형 인터랙션 메트릭
  • 미수집: 비밀번호/결제정보, 키로깅(키 입력 내용), 복사/붙여넣기 "내용", 스크롤·클릭 정밀 좌표

  • 국외이전: 보고서 생성을 위해 미국(Anthropic 본사)로 데이터가 이전(처리위탁)될 수 있음
  • 철회: 언제든 설정에서 비활성화 또는 확장프로그램 제거로 중단 가능
